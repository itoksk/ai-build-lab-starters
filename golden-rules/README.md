# golden-rules — the 7 habits of vibe coding + repair phrases

> Live: https://ai-build-lab.pages.dev/skills/seven-golden-rules

The seven habits that separate a frustrating AI build session from a smooth one (`rules.md`), plus the repair and polish phrases you paste when something breaks or when a "60%" build needs finishing (`repair-phrases.md`). All verbatim from the site.

## Use just this folder

```bash
npx degit itoksk/ai-build-lab-starters/golden-rules my-golden-rules
# or sparse-checkout:
git clone --filter=blob:none --no-checkout https://github.com/itoksk/ai-build-lab-starters.git
cd ai-build-lab-starters
git sparse-checkout init --cone && git sparse-checkout set golden-rules && git checkout
```
