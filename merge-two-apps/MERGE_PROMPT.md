# The merge prompt

This is the prompt from the Merge Two Apps Work, **verbatim** from the site. Send it to your AI, then paste both HTML files after it.

> Live: https://ai-build-lab.pages.dev/works/merge-two-apps

```
Here are two web pages. Merge them into one: keep the visual design of the FIRST and add the main feature of the SECOND. Keep it as one HTML file, and don't break either part.
```

## How to use it

1. Paste the prompt above.
2. Paste all of `app-a.html` (the FIRST — design-rich, feature-light).
3. Paste all of `app-b.html` (the SECOND — design-light, feature-rich counter).
4. The AI returns one HTML file: App A's aurora design wrapped around App B's working counter.
5. If either part breaks, fix it **one change per message** (see `golden-rules/repair-phrases.md`).

Because each app is a self-contained single file, the AI can splice them cleanly. This is exactly how four people build four parts separately and still end up with one project.
