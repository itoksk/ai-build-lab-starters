# Vibe-Coding generating prompts

These are the four "fill-in" build prompts from the Mini Build Work, **verbatim** from the site. Each builds one self-contained HTML file in Gemini Canvas (or any AI that returns code). The `.html` files in this folder are realistic examples of what each prompt produces — open them in a browser to see one running.

> Live: https://ai-build-lab.pages.dev/works/vibe-coding-mini-build

The golden rules: **one change at a time**, start from the smallest version, and use the repair phrase (`golden-rules/repair-phrases.md`) when something breaks.

## The full 4-step workflow (before you send a build prompt)

```
Step 1 — Hearing:
I want to build a page that shows off my love of street basketball. Be my sounding board and interview me about it. Don't write any code yet.

Step 2 — Vocabulary:
Give me 8 adjectives I could use to describe a website's vibe, sorted into: energetic, calm, retro, and futuristic. Plain words, no jargon.

Step 3 — Generate:
Build a single self-contained web page that shows off street basketball. Vibe: energetic, bold, a little retro. Show a hero title, 3 favorite moves, and a hype quote. Make it work as one HTML file.

Step 4 — Iterate (one change per message):
Make the hero title bigger and add a subtle bounce animation when the page loads.
```
