# Code-fix template

A ready-to-use prompt template (verbatim from `/troubleshooting`). Replace the `（…）` parts and send.

> Live: https://ai-build-lab.pages.dev/troubleshooting

```
【現在の問題】
（何が問題なのか具体的に書く）

【期待する動作】
（どう動いてほしいのか書く）

【現在のコード】
```
（コードを貼り付け）
```

【エラーメッセージ（あれば）】
（エラー全文を貼り付け）

上記を修正してください。
```
