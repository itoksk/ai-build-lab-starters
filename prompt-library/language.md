# Language & chat prompts

Short prompts for local AI (Ollama / vibe-local). Change temperature to change how varied the answers are. Verbatim from the `/prompts` page.

> Live: https://ai-build-lab.pages.dev/prompts

```
Say hello in three languages.
```

```
In one sentence, explain what a token is.
```

```
Write a haiku about the sea (5-7-5).
```

```
write a one-line slogan for a robot cafe
```

(Compare temperature 0.2 vs 1.5: stiff vs wild.)

```
Using only my sources, what are the three main ideas, and which source does each come from?
```

(For grounded answers, e.g. NotebookLM.)

```
At temperature 2.0, what happens to a tiny GPT's output?
```
