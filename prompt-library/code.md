# Files & coding prompts

Give these to an AI that can run commands (vibe-local, Claude Code) and it builds and runs them for you. Verbatim from the `/prompts` page.

> Live: https://ai-build-lab.pages.dev/prompts

```
Create a file hello.txt that says: Hello AI from Scratch.
```

```
Write fib.py that prints the first 10 Fibonacci numbers, then run it.
```

```
Make a folder called test with a README.md inside that says Hi.
```

```
List the files in this folder and tell me what each one is.
```

```
Write index.html: a simple page that says "Hello, I'm <your name>" with a big heading, one short intro line, and a button that shows an alert when clicked.
```

```
Write index.html: a one-page profile with my name as an h1, a 2-line bio, and a button that toggles a dark background.
```

```
Commit all my changes with a short, clear message describing what I changed, then push to GitHub. Show me each command before you run it.
```

```
Don't break the existing code structure, keep all comments and logs, and fix only the related part. The problem:
```
