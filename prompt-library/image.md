# Image-generation prompts

English is more stable. Think in four parts: **Subject, Style, Composition, Negative**. Verbatim from the `/prompts` page.

> Live: https://ai-build-lab.pages.dev/prompts

```
A photorealistic dinosaur living in a modern city, doing an ordinary human job
```

```
a cozy ramen shop on a rainy night, neon signs, watercolor style
```

```
A cute mascot character that represents a person who is [curious, warm, loves music]. Flat sticker style, simple background.
```

```
photo of a red apple on a white background
```

## The 4-part build (Subject + Style + Composition + Negative)

"Cool" means nothing to an AI; these four do.

```
[Subject] a weathered rescue robot [Style] Studio Ghibli watercolor [Composition] low-angle, golden-hour backlight [Negative] no text, no logos
```

## Negative prompt (what you DON'T want)

```
blurry, extra fingers, text, watermark
```
